# Budget App

Eine einfache persönliche Budget-Web-App für Einnahmen, Ausgaben, Schulden und Vermögen.

## Wichtig

Die App speichert Daten lokal im Browser (`localStorage`). Besucher sehen nicht deine Daten. Jeder Nutzer hat eigene lokale Daten.

Private Dateien wie Backup-JSON oder Excel-Dateien gehören nicht ins Repository.
